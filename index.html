2 — Cos'è il Kv e perché è importante
Il coefficiente di flusso determina quanta portata passa attraverso una valvola a una data pressione
Formula Kv — sistema metrico europeo
Q = Kv · √ΔP
Q = portata [m³/h]  ·  Kv = coefficiente di flusso  ·  ΔP = perdita di carico [bar]
oppure: ΔP = (Q / Kv)²

Un Kv alto significa che la valvola lascia passare molta portata con poca perdita di pressione. Un Kv basso significa strozzatura: la pompa deve spingere di più per ottenere la stessa portata.

Esempio pratico a 50 l/min (0.83 m³/h): SmartValves Kv 14 → ΔP = (0.83/14)² = 0.0035 bar
Solenoide Kv 2.3 → ΔP = (0.83/2.3)² = 0.13 bar
Il solenoide perde 37× più pressione sulla stessa portata.
